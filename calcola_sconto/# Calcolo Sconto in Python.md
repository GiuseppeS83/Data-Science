# Calcolo Sconto in Python

Questo script Python permette di calcolare il prezzo finale di un prodotto dopo l'applicazione di uno sconto in percentuale.

## Funzionalità

- Calcola lo sconto dato un prezzo e una percentuale di sconto.
- Restituisce il prezzo finale dopo aver applicato lo sconto.

## Requisiti

- Python 3.x

## Uso

1. Salvare il file come `calcola_sconto.py` (o usare il nome originale).  
2. Aprire il terminale e spostarsi nella cartella del file.
3. Eseguire lo script:

```bash
python calcola_sconto.py
