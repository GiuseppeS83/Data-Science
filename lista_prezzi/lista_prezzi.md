# Filtro Prezzi Python

Questo script è una piccola utility scritta in **Python** per filtrare automaticamente una lista di valori numerici in base a una soglia definita.

---

## 📋 Descrizione

Il file `lista_prezzi.py` analizza un dataset di prezzi e genera una sottolista contenente solo i valori che soddisfano i criteri di business.

### Logica del codice:
* **Input**: Una lista predefinita di interi (`prezzi`).
* **Processo**: Un ciclo `for` itera attraverso ogni elemento.
* **Filtro**: Viene applicata la condizione `if prezzo > 50`.
* **Output**: Una nuova lista stampata in console con i soli valori filtrati.

## 🚀 Come utilizzarlo

1. **Prerequisiti**: Assicurati di avere Python (versione 3.x consigliata) installato.
2. **Download**: Salva lo script come `lista_prezzi.py`.
3. **Esecuzione**: Apri il terminale e lancia il comando:
   ```bash
   python lista_prezzi.py
