# 📌 Set per Valori Unici (Python)

## 📖 Descrizione

Questo script Python dimostra come utilizzare la struttura dati **set**
per ottenere valori unici da una collezione di dati.

In Python, un *set* è una struttura che: - Non permette duplicati - Non
mantiene un ordine specifico - È utile per operazioni matematiche tra
insiemi

------------------------------------------------------------------------

## 🎯 Obiettivo dello Script

Lo scopo del file è:

-   Mostrare come eliminare duplicati da una lista
-   Dimostrare la creazione e l'utilizzo dei set
-   Comprendere il comportamento dei set in Python

------------------------------------------------------------------------

## 🛠 Requisiti

-   Python 3.x

Non sono necessarie librerie esterne.

------------------------------------------------------------------------

## ▶️ Come eseguire lo script

1.  Apri il terminale

2.  Vai nella cartella dove si trova il file

3.  Esegui il comando:

    python "Set per valori unici.py"

------------------------------------------------------------------------

## 💡 Concetti Chiave

-   Creazione di un set: `set(lista)`
-   I set rimuovono automaticamente i duplicati
-   Operazioni possibili:
    -   Unione
    -   Intersezione
    -   Differenza
